Lists in HTML

Ordered List

An ordered list is a list that is numbered. Each item in the list is preceded by a number or letter. Example of an ordered list:

  1. First Item
  2. Second Item
  3. Third Item

Unordered List

An unordered list is a list that uses bullet points instead of numbers. Each item in the list is preceded by a bullet point. Example of an unordered list:

  • Apple
  • Banana
  • Cherry

Lists are useful for organizing information in a clear and structured way.
