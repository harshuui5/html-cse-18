Text Formatting in HTML

Bold Text

You can make text bold using the <b> or <strong> tags. Example:

This is bold text using <b> tag.

This is strong text using <strong> tag.

Italic Text

You can italicize text using the <i> or <em> tags. Example:

This is italic text using <i> tag.

This is emphasized text using <em> tag.

Underlined Text

You can underline text using the <u> tag. Example:

This is underlined text using <u> tag.

Strikethrough Text

You can create strikethrough text using the <s> or <del> tags. Example:

This is strikethrough text using <s> tag.

This is deleted text using <del> tag.

Superscript and Subscript Text

You can create superscript text using the <sup> tag and subscript text using the <sub> tag. Example:

This is superscript: E = mc2

This is subscript: H2O
